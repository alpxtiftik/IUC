# IUC – Computer Engineering | Assignments & Projects

This repository hosts the following materials prepared within the scope of the Istanbul University-Cerrahpaşa (IUC) Computer Engineering undergraduate program:

- Course assignments,
- Laboratory applications,
- Small projects
- Practice/trial code.

All content is organized by semester and subject according to the curriculum.
